## [0.1] - 20521106

### Added
- Initial release
- `JSON Search String` keyword for single result queries
- `JSON Search List` keyword for list result queries
- Full JMESPath syntax support via jmespath.py library
- Python 3.10+ support
- Type hints and mypy validation
- Comprehensive test suite
- GitHub Actions CI/CD
